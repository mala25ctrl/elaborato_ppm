# Replica del "Fatto Quotidiano"

Questa è una replica del sito del "Fatto Quotidiano", costruita utilizzando solo HTML, CSS e JavaScript.

La librerie utilizzata è [Bootstrap](http://getbootstrap.com/).

Esso è stato realizzato come compito universitario per il corso **B028451 (B047) - PROGETTAZIONE E PRODUZIONE MULTIMEDIALE** dell'Università di Firenze.

Tutti i contenuti sono fittizi, solo la struttura e il layout sono stati replicati.
